### 🐍 Python 科学计算与后处理工具包 (Scientific Computing & Post-Processing)

* 🚀 **量子化学与第一性原理自动化**：集成 **VASP** 与 **CP2K** 的任务程序化管理工具，实现批量作业提交、状态监控与集群任务链自动化。
* 📊 **综合性 DFT 计算后处理**：专为电荷密度、态密度（DOS）、能带结构及电子局域函数（ELF）开发的高效数据解析与绘图工作流。
* 🤖 **AI for Science (AI4S)**：定向靶向爬取与高通量筛选指定晶体结构的自动化程序，加速新型材料发现。
* 🪐 **拓扑磁性与多尺度模拟**：**斯格明子（Skyrmion）**动力学行为的可视化模拟工具，以及微观结构演化的**相场模拟（Phase-Field）**通用程序。
* ⚡ **电化学数据高阶分析**：
  * **XPS（X射线光电子能谱）** 自动分峰与定性定量分析工具。
  * **dV/dQ（微分容量）** 曲线转化与相变路径解析程序。
  * **DRT（弛豫时间分布）** 算法模型，用于深挖阻抗谱（EIS）中的动力学速控步骤。
* 🎨 **科研可视化**：定制化的学术图表**颜色主题构建工具**，确保发表级图片的美观与规范。
